# 🔒 BrowserHardening

> **Browser Security Hardening** - Applies security policies and hardening to web browsers.

---

## 📋 Overview

BrowserHardening applies comprehensive security policies to Chrome, Firefox, Edge, and other browsers to enhance security, privacy, and prevent common web-based attacks.

---

## 🎯 What It Does

- 🔒 **Security Policies** - Browser hardening configurations
- 🚫 **Dangerous Features** - Disable risky browser functions
- 🔐 **Safe Browsing** - Enhanced protection settings
- 🛡️ **Extension Controls** - Manage extension permissions
- 🌐 **Multi-Browser** - Chrome, Firefox, Edge support

---

## 🚀 Usage

```powershell
# Harden all browsers
.\BrowserHardening.ps1
```
